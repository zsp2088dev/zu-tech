## ビルドとセットアップ

``` bash
# パッケージのインストール
$ yarn install

# 開発環境で実行 (localhost:3000)
$ yarn dev

# ビルドして実行
$ yarn build
$ yarn start

# アプリケーションをビルドして静的ファイルを生成
$ yarn generate
```
